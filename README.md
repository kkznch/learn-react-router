# Learn React without create-react-app

## Getting Started
### Prerequisites
- yarn
  - https://classic.yarnpkg.com/en/

## Installation
1. Clone the repository
    ```sh
    $ git clone git@github.com:kkznch/learn-react-router.git
    $ cd learn-react-router
    ```
2. Run yarn install
    ```sh
    $ yarn install
    ```

## Usage
Run yarn dev command.
```sh
$ yarn dev
```
